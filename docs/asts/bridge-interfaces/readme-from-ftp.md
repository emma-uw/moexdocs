# Bridge Interfaces — FTP source

ASTS Bridge interface HTML specifications are published on MOEX FTP under `ClientsAPI/ASTS/Bridge_Interfaces/`.

Use the ingest scripts in the repository (`scripts/pull_ftp.py`, `scripts/html_to_md.py`) to refresh Markdown in this site from the FTP tree.

For a machine-readable coverage overview of converted pages, see the `bridge-interfaces-coverage.md` file in the repository root.
