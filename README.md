# MOEX Docs

Docusaurus documentation site with English and Russian support.

## Structure

- `docs/` — English documentation (default locale)
- `i18n/ru/docusaurus-plugin-content-docs/current/` — Russian documentation
- Same file structure in both locales: intro, getting-started, guide

## Commands

- `npm run start` — Dev server (http://localhost:3000)
- `npm run build` — Production build
- `npm run serve` — Serve build locally
- `npm run write-translations -- --locale ru` — Regenerate i18n JSON for Russian

## Locales

- **en** — English (default)
- **ru** — Russian

Use the language switcher in the navbar to switch locales.
