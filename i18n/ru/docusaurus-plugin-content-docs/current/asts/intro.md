# ASTS

В этом разделе MOEX Docs приведён справочник по шлюзовым интерфейсам ASTS Bridge — интерфейсам, с помощью которых участники подключают свои системы к рынкам Московской биржи.

## Что такое ASTS?

**ASTS** (Автоматизированная торговая система) — платформа торговли и клиринга, используемая Московской биржей для фондового, валютного и других рынков. Она обеспечивает маршрутизацию заявок, рыночные данные и расчётные процессы для брокеров и других участников.

## Что такое ASTS Bridge?

**ASTS Bridge** — пакет для подключения, позволяющий участникам интегрировать своё ПО с ASTS. В его состав входят:

- спецификации интерфейсов, описывающие структуры, сообщения и протоколы системы;
- демо-приложение **ASTS Client**, с помощью которого можно изучить структуру интерфейсов и сформировать описания для используемой версии.

Спецификации публикуются в информационных целях. Аналогичные описания для вашей версии интерфейса можно получить через функцию **MTEStructure** в вашем ПО или экспортом структуры из демо ASTS Client.

## Шлюзовые интерфейсы и использование только брокерских

Необходимо использовать только **брокерские** интерфейсы. При подключении укажите интерфейс в параметре соединения `INTERFACE` одним из следующих вариантов:

| Значение INTERFACE | Назначение |
|--------------------|------------|
| `IFCBrokerXX` | Торговые системы валютного и фондового рынков (см. спецификации Currency и Equities) |
| `IFCBrokerXXR` | Клиринг валютного рынка (см. спецификации RiskCur) |
| `IFCBroker_XX` | Рынок депозитно-кредитных операций (см. спецификации GKO) |
| `IFCAuxBrkXX` | Вспомогательные функции торговой системы фондового рынка (см. спецификации Auxiliary) |

Здесь `XX` — номер версии интерфейса.

## Версионирование

Публикуемые спецификации соответствуют поддерживаемым в данный момент производственным версиям. При интеграции рекомендуется использовать последнюю поддерживаемую версию.
