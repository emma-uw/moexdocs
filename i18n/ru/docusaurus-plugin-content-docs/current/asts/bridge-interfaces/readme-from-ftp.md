# Bridge Interfaces — источник на FTP

HTML-спецификации интерфейсов ASTS Bridge публикуются на FTP МосБиржи в каталоге `ClientsAPI/ASTS/Bridge_Interfaces/`.

Для обновления Markdown на сайте из дерева FTP используйте скрипты репозитория (`scripts/pull_ftp.py`, `scripts/html_to_md.py`).

Обзор покрытия конвертированных страниц см. в файле `bridge-interfaces-coverage.md` в корне репозитория.
