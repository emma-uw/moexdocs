| Константа | Значение | Описание |
| --- | --- | --- |

#### TQuoteStatus - Статус квоты (длина: 1, тип: ekGroup)

Перечень возможных значений статуса квоты

| Константа | Значение | Описание |
| --- | --- | --- |
| O | Акт | Активная |
| W | - | Снята |
| R | -Т | Отклонена торговой системой |
| C | -С | Снята торговой системой |

| I | Ввод встречных | Ожидает ввода встречных адресных заявок |
| J | Завершена | Неактивна (снята) при наличии удовлетворенных заявок |

| Константа | Значение | Описание |
